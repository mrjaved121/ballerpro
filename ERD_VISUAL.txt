                              RELATIONSHIP SUMMARY
═══════════════════════════════════════════════════════════════════════════════

ONE-TO-MANY:
  User → Workout (createdBy)
  User → WorkoutSession
  User → Meal
  User → Post
  User → BodyMeasurement
  User → ProgressPhoto
  User → ProgramEnrollment
  Workout → WorkoutSession
  Program → ProgramEnrollment
  Post → PostLike (embedded)
  Post → PostComment (embedded)

MANY-TO-MANY:
  User ↔ User (via Follow)
  Workout ↔ Exercise (via WorkoutExercise)
  Program ↔ Workout (via ProgramWorkout)

EMBEDDED DOCUMENTS:
  WorkoutExercise (in Workout)
  PostLike (in Post)
  PostComment (in Post)
  Onboarding (in User)
