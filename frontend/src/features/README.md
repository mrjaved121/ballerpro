# Features

Feature-based organization for business logic and feature-specific components.

## Structure

Each feature directory can contain:
- `components/` - Feature-specific components
- `hooks/` - Feature-specific hooks
- `api.ts` - API calls for the feature
- `types.ts` - TypeScript types for the feature
- `utils.ts` - Feature-specific utilities

## Features

- `auth/` - Authentication feature
- `workouts/` - Workouts and training feature
- `programs/` - Training programs feature
- `community/` - Community/social feature
- `shop/` - E-commerce/shop feature
- `tracking/` - Progress tracking feature
